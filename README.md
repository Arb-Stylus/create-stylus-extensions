# 🔌 create-stylus Extensions

This repository holds all the BG curated extensions for [create-stylus](https://github.com/Arb-Stylus/create-stylus), so you can extend the functionality of your Scaffold-Stylus project.

## Usage

You can install any of the extensions in this repository by running the following command:

```bash
npx create-stylus@latest -e <extension-name>
```
